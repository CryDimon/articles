# Руководство пользователя <a href="http://kato.im" target="_blank">Kato.IM</a> и документация

---

Исходные тексты всех документов руководства представлены на GitHub: <a href='https://github.com/kato-im/articles' target='_blank'>http://github.com/kato-im/articles</a>. Мы будем очень благодарны, если вы, наш пользователь, внесёте вклад в описание мессенджера Kato и его возможностей. В нижней части каждой статьи есть линк "Предложить улучшение", используя который можно легко создать новый pull request с вашими дополнениями и исправлениями:
